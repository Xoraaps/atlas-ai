# Atlas AI

> Open-source AI companion with memory, planning, learning, finance, and multi-agent capabilities.

Atlas AI is an experimental open-source project that aims to build a persistent AI companion capable of helping users manage knowledge, goals, learning, finances, projects, and daily life through a unified memory and reasoning system.

---

## Vision

Current AI systems are powerful but fragmented.

Users interact with multiple tools for:

- Learning
- Productivity
- Knowledge Management
- Finance
- Project Planning
- Personal Growth

Atlas AI aims to unify these experiences into a single intelligent companion.

---

## Core Principles

### Persistent Memory

Remember important information over time.

### Context Awareness

Connect information from multiple domains.

### Multi-Agent Architecture

Allow specialized agents to collaborate.

### User Ownership

User data remains portable and transparent.

### Open Source

Built publicly and collaboratively.

## Planned Features

### Memory Engine
